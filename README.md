# MidiMonster Wiki

### Initial Setup

```shell
$ virtualenv venv --python=python3.7
$ source venv/bin/activate
$ pip install -r requirements.txt
```

### Building

```shell
$ source venv/bin/activate
$ make html
```

The HTML views are now present in `build/html`.
